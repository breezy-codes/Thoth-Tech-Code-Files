# Usage example creation for Validate Double Input

This is a usage example for validating whether user input is a valid double using SplashKit. The program prompts the user to enter a value and then checks if the input can be interpreted as a double, displaying the appropriate message. The PR is [here](https://github.com/thoth-tech/splashkit.io-starlight/pull/999)
