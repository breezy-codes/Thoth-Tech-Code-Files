# Usage example creation for Finding Word Index

This is a usage example for finding the index of a specified word within a sentence using SplashKit. The program prompts the user to enter a sentence and the word to find, then outputs the index of the first occurrence of that word. The PR is [here](https://github.com/thoth-tech/splashkit.io-starlight/pull/999)
